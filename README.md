## Installation
Just add this line to your `sitemap.xml`:

```xsl
<?xml-stylesheet type="text/xsl" href="https://gitcdn.xyz/cdn/pedroborges/xml-sitemap-stylesheet/master/sitemap.xsl"?>
```
